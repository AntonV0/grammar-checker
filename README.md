# Grammar Checker
## Project #32 from Codecademy's Full Stack Engineer Career Path
### By Anton Vlasenko (AntonV0)  
## Project Outline
The project's goal is to fix the grammatical errors of a short story. This included removing the unnecessary word "literally", removing words with over 10 characters, changing the misspelled word "beautifull", and removing the bad word "freaking".
## What I Learned
  - Using various array methods such as forEach(), filter(), map(), findIndex(), every(), and join()
